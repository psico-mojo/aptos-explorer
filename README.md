## How to use

Clone the repo.
If you are running node.js ; install yarn via terminal or
 Requires [pnpm](https://pnpm.io/installation)

Then ; 
Install dependencies:
```sh
pnpm install
```

Run below to start the app:

```sh
pnpm start
```
note; view on dev server @ localhost 3000
